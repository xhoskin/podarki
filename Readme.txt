Для начала работы:
1) скоприровать проект в рабочую папку, переназвать
2) зайти в папку с проектом по SSH
3) Изменить права всем папкам и файлам на 0777
- sudo chmod 0777 -R *
4) Установить необходимые библиотеки
- npm install

#####################################################################################################

Описание комманд:
- grunt sprite
Сбор картинок из папки /source/images/sprites в спрайт /web/images/sprite.png
Создание _sprites.less файла с переменными о картинках в спрайте:
@{имя файла}-image    - путь до спрайта
@{имя файла}-width    - ширина исходной картинки
@{имя файла}-height   - высота исходной картинки
@{имя файла}-offset-x - позиция картинки в спрайте по оси X
@{имя файла}-offset-y - позиция картинки в спрайте по оси Y

- grunt less
Компиляция source/less/style.less в /web/css/style.css

- grunt jade
Компиляция /source/*.jade файлов в /web/templates/*.html
Компиляция сводной страницы /source/jade/all.jade в /web/index.html
Для указания названия страницы используется первая строка файла /source/*.jade

- grunt sync
Синхронизация файлов в папках
/source/images/* -> /web/images/*
/source/javascript/lib/*.js -> /web/js/lib/*.js

- grunt prettify
Форматирует /web/templates/*.html по заданным правилам

- grunt push
Выполняет полный цикл комманд описанных выше

- grunt imagemin
Оптимизирует картинки в папке /web/images/*

- grunt clean
Удаляет .svn папки из проекта

- grunt lite
Компилирует less, js, jade и images

#####################################################################################################

Описание less-миксинов:

.make-icon(@name) - создает класс .icon-<@name>, он содержит фон, ширину и
высоту картинки. Лежит в файле less/icons.less
